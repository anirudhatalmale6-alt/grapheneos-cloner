PLATFORM TOOLS
==============

This directory should contain the following files from Android SDK Platform Tools:

  - adb.exe          (Android Debug Bridge)
  - fastboot.exe     (Fastboot flashing tool)
  - AdbWinApi.dll    (ADB Windows API library)
  - AdbWinUsbApi.dll (ADB Windows USB API library)

Download the latest platform-tools from:
https://developer.android.com/tools/releases/platform-tools

Extract and copy the files listed above into this directory.

The application will use these bundled binaries so the end user
does not need to install Android SDK separately.
